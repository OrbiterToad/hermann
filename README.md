<h1>Hermann</h1>
<p>Applicaton to Control Computers Remote via Commandline and Custom Commands</p>
<hr>
<h3>Support</h3>
<h4>tested</h4>
<li>- Windows 7</li>
<li>- Windows 10</li>
<h4>not tested</h4>
<li>- Windows xp</li>
<li>- Windows Vista</li>
<li>- Windows 8</li>
<li>- Linux</li>
<h3>Features</h3>
- [x] Client Information (IP, User, OS, etc.)<br>
- [x] Place in Autostart<br>
- [x] Print Files<br>
- [x] Download Files<br>
- [x] Chat with Client<br>
- [x] Screenshot<br>
- [x] Webcam<br>
- [x] Play Sounds<br>
- [x] Control Tasks<br>
- [ ] Run VB Scripts<br>
- [ ] Keylogger
